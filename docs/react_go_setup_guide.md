# React (TypeScript) & Go セットアップガイド

MasonとLSPの設定により、ReactとGoの開発環境が整いました。

## インストールされたツール (LSP)

`lsp.lua` によって以下のLSPサーバーが自動的にインストール・設定されます。

| 言語/フレームワーク | サーバー名 | 役割 |
| :--- | :--- | :--- |
| **TypeScript / React** | `ts_ls` | コード補完、定義ジャンプ、型チェック |
| **JavaScript / TS** | `eslint` | リント（構文チェック）、保存時の自動修正 |
| **Go** | `gopls` | コード補完、定義ジャンプ、ドキュメント参照 |

## 追加でインストール推奨のツール

LSP以外のフォーマッターやリンターは、Masonを使って手動でインストールすることをお勧めします。

### インストール手順

1. Neovimで `:Mason` を実行してウィンドウを開く。
2. 以下のツールを検索 (`/` キー) し、インストール (`i` キー) してください。

#### 1. Prettier (React/TSフォーマッター)

- パッケージ名: `prettier`
- 用途: コードの自動整形

#### 2. Go Tools

- パッケージ名: `golangci-lint` (リンター)
- パッケージ名: `goimports` (インポート整理・フォーマッター)

## 動作確認

### React / TypeScript

1. `.tsx` または `.ts` ファイルを開く。
2. `:LspInfo` を実行し、`ts_ls` と `eslint` が attatch されていることを確認。
3. 適当なコードを書いて補完やエラー表示が出るか確認。
4. 保存時にESLintの自動修正が走るか確認。

### Go

1. `.go` ファイルを開く。
2. `:LspInfo` を実行し、`gopls` が attatch されていることを確認。
3. コード補完が効くか確認。
