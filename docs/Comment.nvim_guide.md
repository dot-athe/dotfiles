# Comment.nvim セットアップガイド

## ドキュメント

- [GitHub Repository](https://github.com/numToStr/Comment.nvim)

## 概要

**Comment.nvim** は、行やブロックのコメントアウトを強力にサポートするプラグインです。
「現在行をコメントアウト」「選択範囲をコメントアウト」といった操作を、直感的なキーバインド (`gc`, `gcc` 等) で提供します。

## 前提条件

- **NeoVim Version**: `v0.5` 以上
- **Lazy.nvim**: プラグインマネージャー

## 設定例 (lazy.nvim)

`lua/plugins/comment.lua` を作成し、以下のコードを記述します。

```lua
return {
  "numToStr/Comment.nvim",
  event = { "BufReadPost", "BufNewFile" },
  opts = {
    -- ここにオプションを記述（デフォルトで十分便利な設定になっています）
  },
}
```

## 基本操作一覧

### ノーマルモード

| キー | アクション | 説明 |
| :--- | :--- | :--- |
| `gcc` | トグル (行) | カレント行のコメントを切り替え |
| `gbc` | トグル (ブロック) | カレント行のブロックコメントを切り替え |
| `gc` + 移動 | トグル (範囲) | 移動範囲をコメント切り替え (例: `gc2j` で下2行) |
| `gco` | 挿入 (下) | 下の行にコメントを挿入してインサートモードへ |
| `gcO` | 挿入 (上) | 上の行にコメントを挿入してインサートモードへ |
| `gcA` | 挿入 (行末) | 行末にコメントを挿入してインサートモードへ |

### ビジュアルモード

| キー | アクション | 説明 |
| :--- | :--- | :--- |
| `gc` | トグル (選択) | 選択範囲を行コメント切り替え |
| `gb` | トグル (ブロック) | 選択範囲をブロックコメント切り替え |
